# Briefing do Projeto

## Dores do Cliente

- Dificuldade em decidir entre acordo ou processo para casos legais, levando a resultados subótimos.
- Incapacidade de prever com precisão o resultado de um processo (extinção, improcedência, procedência parcial, procedência, acordo).
- Falta de uma política clara e baseada em dados para a tomada de decisões de acordo.
- Dados existentes (planilha) são insuficientes ou de baixa qualidade para análises complexas.
- Conhecimento específico do advogado não está capturado nos dados, levando a lacunas nas decisões baseadas apenas em dados.
- Dificuldade em garantir que os advogados sigam a política de acordos definida.
- Necessidade de resultados probabilísticos, não apenas binários (acordo/processo).

## Escopo Técnico

Desenvolvimento de um sistema baseado em IA para auxiliar na decisão entre acordo e processo para casos legais, utilizando uma arquitetura multiagente. O projeto inclui a criação de um modelo/agente para simular e otimizar a política de acordos do banco, um copiloto para advogados e um dashboard para visualização. A política de acordos será um conjunto de regras (árvore de decisão) gerado e otimizado pelo sistema, fornecendo uma recomendação com probabilidade e justificativa. A solução deve ser adaptável a diferentes tipos de processos e incorporar análise histórica (jurisprudência).

## Prazos e Milestones

- KICKOFF
- Simulação da política do banco (desmodelagem)
- Desenvolvimento do copiloto para advogados
- Entrega do Dashboard
